# transactionId

[androidJvm]\
val [transactionId](transaction-id.md): [String](https://kotlinlang.org/api/core/kotlin-stdlib/kotlin/-string/index.html)? = null

Correlation ID to refer to this transaction.

This field is optional. It is generated by the merchant and is used for referring to this transaction later on (e.g. for debugging issues when communicating with Google).
