# showsAsPrimaryButton

[androidJvm]\
val [showsAsPrimaryButton](shows-as-primary-button.md): [Boolean](https://kotlinlang.org/api/core/kotlin-stdlib/kotlin/-boolean/index.html) = true

If true, Google Pay is shown as a standalone button on top of the payment sheet. If false, Google Pay appears as a regular payment method option.
